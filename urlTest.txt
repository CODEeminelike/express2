2. Xử lý đánh giá nhà hàng
text
POST   /api/restaurants/:restaurantId/reviews/:userId    # Thêm đánh giá
GET    /api/restaurants/:restaurantId/reviews            # Lấy danh sách đánh giá theo nhà hàng
GET    /api/users/:userId/restaurant-reviews             # Lấy danh sách đánh giá theo user
3. User đặt món
text
POST   /api/orders/:userId                               # Thêm order
